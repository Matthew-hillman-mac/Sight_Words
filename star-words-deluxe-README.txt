Star Words Deluxe

Open star-words-deluxe.html directly from Files, Downloads, or a browser tab.
It is a single self-contained HTML file with all CSS and JavaScript inline.

Included features:
- Child profiles
- 10-word missions
- Quick retry round for missed words
- Spaced review scheduling
- K / 1st / 2nd grade built-in word bank
- Theme switch: Unicorn or Dino
- Offline browser speech
- Optional mic check when browser supports it
- Local-only save data
- JSON backup export/import
- CSV progress export
- Custom word import from .txt or .csv
- Treats, mascot feeding, and sticker shelf rewards

Notes:
- Best opened in Safari/Chrome/Edge, not a raw text viewer.
- For the most reliable offline speech, install a local system voice on the device.
- On iPhone/iPad, open in Safari and use Add to Home Screen if you want an app icon.
